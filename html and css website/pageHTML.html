    Introduction to Elements

    HTML (Hyper Text Markup Language) is the standard markup language used to create the stucture and content of a webpage.
    HTML is made up of elements, which include tags, attributes and content. These elements are understood by a web browser to create a webpage.

    Below are some of the more common tags used within HTML.

    <p> - Paragraph, used to surround text.
    <img>, - Image, to add an image to HTML.
    <video> - To add a video to HTML.
    <div> - Division, to divide elements into groups which can be styled together from CSS.

    Most tags require a closing tag, indicating the end of the content contained within the tag.
    <p>This text is inbetween a opening and closing p tag.</p>
    However, some tags are self closing, such as the image and video tags.
    The below is an example of an image tag, which as you can see closes itself after the required attributes have been entered.
    <img src="URL source of the image" alt="Text to be displayed if image isn't available. />

    Document Structure

    The first line of an HTML document is <!DOCTYPE html>, which tells the browser that the document is HTML.

    An HTML document is made up of a <head> and <body> section.

    The head section of the HTML page contains meta data and settings. Anything which is not displayed on the webpage itself.
    The <title> for example, is stored within the head section. This tells the browser what to display within the tab holding the webpage. The title of this webpage is HTML and CSS.
    The body section of the HTML document includes anything which is to be displayed to the user. The section you are reading now is part of the body of this webpage.

    Divisions

    The <Div> tag is used to divide different sections of the HTML. The ID element of the <Div> tag is used to give an identifier to the section.
    The <Div> tag should only be used for styling purposes. Structuring a website should instead be achieved using semantic elements.

    Semantic Elements

    Semantic Elements allow for structuring of a webpage, and allow for search engines (and other tools) to more accurately determine the components of a webpage.
    Some semantic elements have existed in HTML since HTML's inception, however the introduction of HTML 5 in 2008 included the majority of the semantic tags now in use.

    Before HTML 5, many developers relied on the use of Div tags to define the components of a webpage.
    This leads to difficulties for search engines, web scrapers, and other tools to determine a websits contents as the ID of a div tag is defined by the developer.

    Some of the primary semantic elements used today are listed below.

    • <header> element represents a container for introductory content or a set of navigation elements. Not to be confused with the <head> element, used to a webpages meta deta.
    • <nav> is used to define a block used for navigation, such as menus or a table of contents.
    • <main> is used to hold the primary contents of a webpage.
    • <footer> is used to hold the contents at the bottom of a webpage, such as contact information, copyright information, terms of use, etc.
    • <section> defines elements in a document, such as chapters, headings or any other areas of the document with the same theme. A websites homepage could be split into sections for the introduction, news items and contact information for example.
    • <article> holds content which makes sense on its own, such as articles, blogs, comments.

    Using HTML structure, these semantic elements can be parents for other semantic elements.
    All articles of a certain type may exist within a section. (Parent section, child article)
    Articles may be made of sections, such as introduction, chapters, contact information (Parent article, child section).

    Text Manipulation

    Note that the resulting text displayed to the webpage can be altered using CSS, including the behaviour of these tags.

    6 different headings can be used in HTML, <h1> through to <h6>.

    The <strong> tag can be used to make text bold.
     The <em> tag can be used to emphasise text, usually in italics.

    Images

    Images are added to an HTML page using the <img> tag.
    Images are not technically inserted into a web page, but linked. The <img> tag creates a holding space for the referenced image.
    The <img> tag includes two required attributes.


  • src - Specifies the path of the image.
  • alt - Specifies the alternate text of the image.
    •

    The src attribute includes a URL path to an image. Images stored internally or externally can be linked.
    The alternate text is used if the image can't be found, if the image can't be displayed for some other reason, or if the user has a screen reader.

    The size of an image can be set by using the width and height attributes, or by using the style attribute.
    <img src="Image URL" alt="Alternate Text" width="128" height="128">
    <img src="Image URL" alt="Alterate Text" style="width:128px;height:128px">
    Note that often, the size of an image is controlled by CSS.
